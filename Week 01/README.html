Week 01 - Roll The Dice

This is your first lab. This is a two-part lab. In the first part of the lab, you will enter some data into Excel and perform some simple calculations on the data. In the second part of the lab, you will perform the functionally equivalent tasks in R.

The goal of this lab is two-part. I want you to get a better idea of how R and Excel are broadly equivalent. Some things are easier in R. Some things are easier in Excel. Excel is more visual, but combines data storage, management, and analysis. As you will see, R separates data storage, management, and analysis.

This is also an oportunity to roll some dice and create some random samples which we will (hopefully) use later in the course to demonstrate p-hacking.

Resources

To complete the Excel portions of this lab, you will need to know how to enter data into Excel, and to apply formulas to small data sets. If you are not familiar with Excel formulas, the following resources will be useful.

  • Overview of Formulas in Excel
  • Calculate Average In Excel (YouTube)
  • Relevant Excel functions/commands:
    • Mean (Average) = avg
    • Sum = sum
  • Relevant R functions/commands:
    • Mean = mean
    • Sum = sum
    • To get help on a function in R, you can prepend the function name with a question mark. For example ?sum will get you the documentation for the sum function.

As with all things programming, Google is your friend. And, XKCD provides us with additional explanation.

Lab Tasks

Part One

Data and code in a spreadsheet program such as Excel, LibreOffice, Google Sheets, etc. is stored in a workbook. Within a workbook, there can be multiple worksheets, which are similar to having multiple tabs in a browser.



  1. Download the lab workbook, here.
    • On the Read Me worksheet, there are some additional in-line instructions.
  2. On the Data worksheet, fill-in the missing values for Initials and Favorite Color.
    • Initials: Since my name is Andy Choens, I would enter AC.
    • Favorite Primary Color: Should be either Red, Green, or Blue. Spelling and capitalization matter.
    • All 15 rows should have the same entries for the first two columns.



  1. Use the two dice provided in class to roll 15 pairs of results. As for which die is Die 1 and which is Die 2 is up to you.
    • The only valid entries for columns D and E are integers between 1 and 6.
  2. Row Sum: Calculate the sum of the two dice, for each roll R01 - R15. You can do this with a formula using either the + operator or sum function.
  3. Row Mean: Calculate the average of each roll R01 - R15. The average of each roll is \((Die 1 + Die 2) / 2\) Please note that becase of the headers, the fifteenth row of data is on the sixteenth row of the worksheet.
